🌍 Kuzana — Hidden Champions Discovery Engine
"Kenya's best businesses are busy building. Our job is to find them."
What is this?
A full AI-powered platform that discovers, profiles and tracks Kenya's Hidden Champions — profitable, unglamorous SMEs that the startup ecosystem has completely missed.
Built for Bounty 5 of the Kuzana programme.
What it does
Tab
Function
🤖 AI Finder
Searches real web sources (directories, Google Maps, news, tenders) to find hidden businesses by sector and county
🏆 Top 50
Tracks the 50 best businesses with full profiles, scores and relationship status
📱 Field Scout
Mobile form for on-the-ground business logging — AI builds the full profile automatically
🤝 Refer & Earn
Scout registration + referral tracking (Ksh 50,000 reward per successful investment)
📖 Methodology
Full documented discovery process — repeatable by anyone
How to run it
This is a React single-file app that runs as a Claude.ai artifact.
Option A — Run in Claude.ai (easiest, works today)
Go to claude.ai
Start a new conversation
Upload kuzana-v3.jsx
Ask Claude: "Run this as an artifact"
Done ✅
Option B — Deploy as a real website
You will need:
Node.js installed
An Anthropic API key from console.anthropic.com
A Next.js or Vite project to wrap the component
